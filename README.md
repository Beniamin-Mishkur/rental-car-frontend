# RentalCar Frontend

Цей проєкт — реалізація фронтенд частини веб‑додатку для компанії **RentalCar**, яка надає послуги оренди автомобілів. Він написаний за допомогою **Next.js** та **TypeScript** та використовує відкритий API для отримання інформації про автомобілі.

## Основні функції

- **Домашня сторінка** з банером та закликом до дії, а також коротким описом переваг сервісу.
- **Каталог автомобілів** з можливістю фільтрації за брендом, максимальною ціною та пробігом. Список автомобілів завантажується сторінково, є кнопка «Load more» для підвантаження наступної порції даних.
- **Сторінка детального опису** окремого автомобіля з великою фотографією, характеристиками, списком аксесуарів та функціональностей, а також формою для бронювання.
- **Система обраного** — користувач може додавати автомобілі до обраних (серце), перелік зберігається у `localStorage` та не зникає при оновленні сторінки.
- Управління станом здійснюється через **Zustand**. Запити до бекенду — через **Axios**.

## Структура проєкту

- `src/app` — папка зі сторінками (Next.js App Router). Тут знаходяться домашня сторінка (`page.tsx`), сторінка каталогу (`/catalog/page.tsx`) та сторінка детального опису (`/catalog/[id]/page.tsx`).
- `src/components` — багаторазові компоненти: картка автомобіля, панель фільтрів, кнопка обраного, форма бронювання.
- `src/store` — глобальний стан застосунку на базі Zustand (`useCarsStore`).
- `src/types` — оголошення TypeScript типів для автомобілів.
- `public/images` — статичні зображення (фонове зображення для домашньої сторінки).
